An app for tracking orders, items and procedures in a machine shop.

The app stores informations about orders, items, dates of coming in- getting out, procedures done to items with details and pictures.

The possible procedures are managed by the admin/manager and they help suggest a price for every procedure done to a type of item for easier work-flow. Also the procedures have a rank to help manage the order of logic operations.

An order can have multiple items of 4 types. For the items you can assign possible procedures for that type, add details and before-after photos.

Each procedure will be tracked to know who completed it for a better understanding of the personnel in the machine shop.

There is a tab in the app to see the next procedure in line, waiting to be completed, for each item, so no item will be forgotten. Also an user might be specialised on a specific procedure, they can look for that procedure waiting to be done, for any ready item in line.
